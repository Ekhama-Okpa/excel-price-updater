# excel-price-updater
Updating the prices of produce in an Excel file using the openpyxl module on Python

This project uses the openpyxl module on Python to scan the contents of an Excel file.
It then updates the prices of produce found in column "A" according to the contents of a dictionary.
The keys are the produce and the values are the prices.
The prices are found in the corresponding column "B" and are modified according to the new prices present in the dictionary.
